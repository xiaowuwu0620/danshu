# 丹书 Delivery Checklist

Before final delivery, verify:

- PDF page count matches the user's request or your stated contract.
- HTML contains no unresolved template placeholders.
- HTML/PDF do not contain the old Kami ink-blue token.
- Vermilion is used as focus, not as decoration everywhere.
- Author/class/metadata is present and not visually louder than the title.
- All numbers from the source are preserved accurately.
- No invented data, sources, or dates.
- Curves do not create false associations.
- Numbered dots stay physically paired with labels.
- Right edge and footer have at least 6-8mm safe space.
- Page PNGs are separate files when the user asks for images.
- High-resolution PNGs are 2K-4K on the long edge when requested.
- Final answer links PDF, HTML source, and individual PNGs if produced.
