# 丹书 Danshu

**朱砂红中国风 A4 报告排版 skill。**

丹书把 Kami 的克制纸感改造成更适合中文报告、讲义、调研说明和方案书的视觉语言：暖纸底、朱砂红、宋式层级、曲线图表、负面空间。

> Warm parchment. Vermilion focus. Curved judgment.

## Why It Exists

Most AI-generated reports look like either corporate templates or over-decorated posters. 丹书 aims for a different register:

- Chinese editorial rhythm, not slide-factory gloss.
- Data and judgment before adjectives.
- Curves that explain structure, not decoration.
- Vermilion as a precise focus color, never a red flood.
- PDF and high-resolution page images ready for sharing.

## What It Makes

- A4 portrait PDF reports
- One-page briefs
- Fieldwork / class assignment handouts
- Research summaries
- Proposal-style mini reports
- Page-by-page 2K-4K PNG exports

## Install

Copy the skill folder into your Codex skills directory:

```bash
mkdir -p ~/.codex/skills
cp -R skill/danshu ~/.codex/skills/danshu
```

Then ask Codex:

```text
Use $danshu to turn this PDF into a 3-page A4 Chinese report with curved charts and vermilion focus color.
```
